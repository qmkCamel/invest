# 2025年真实财报案例课

这套文章不是五篇“公司好不好”的结论式研报，而是一套用真实财报训练投资判断的案例课。核心目标是学会三件事：

1. 从业务事实还原利润是怎么来的；
2. 用资产负债表和现金流量表验证利润质量；
3. 把会计结果转换成未来投资需要回答的问题。

资料基准日为2026年7月12日。由于多数公司尚未披露2026年半年报，全文统一以2025年审计年报为数字基准，并用2023-2025年数据观察趋势。等2026年半年报发布后，可以直接按每篇末尾的检查表更新。

## 五家公司为什么放在一起

它们代表五种完全不同的经济模型：

- 贵州茅台：先收钱、低资本开支、品牌定价；
- 美的集团：采购、生产、库存、渠道和全球化共同驱动；
- 中国神华：商品价格和重资产再投资决定正常利润；
- 招商银行：存款是主要“原材料”，贷款是主要“产品”；
- 宁德时代：增长、技术、扩产和质保责任必须同时分析。

真正的读表能力，不是记住“现金流高于净利润就是好”，而是知道为什么同一个指标在不同行业含义不同。

## 文章中的三类表述

为避免把观点伪装成事实，文章统一使用以下区分：

- **报表事实**：可以直接在2025年年报中找到的数字或披露；
- **分析解释**：根据报表之间关系作出的推断；
- **待验证问题**：需要用下一期报告、行业数据或公司经营数据继续验证。

## 数字口径

- 金额默认按人民币十亿元四舍五入，具体表格另有说明的除外；
- 增速、毛利率、净息差等按公司披露口径；
- 中国神华2023-2024年采用2025年报重述数据；
- “自由现金流代理值”统一采用`经营活动现金流量净额 - 购建长期资产支付的现金`，只用于教学，不等同于严格估值口径的股权自由现金流；
- 招商银行不使用普通企业自由现金流，因为存贷款、金融投资和融资现金流本身就是主营业务。

## 使用建议

每篇先读“核心判断”，再自己完成表格和练习，最后看文章解释。不要先看股价，也不要在第一次阅读时追求把所有附注读完。目标是先建立业务与数字之间的因果关系，再逐步深入会计估计和估值。

> 这套材料用于财报学习，不构成个股推荐、收益承诺或个性化投资建议。好公司、好财报和好买入价格是三个不同问题。
